# Contributing to TORDEN

This is a family repo — all skill levels welcome. 🎉

## Adding a new project

1. Copy `templates/project-template/` → `projects/NN-your-project-name/`
2. Fill in the `README.md`
3. Add a row to the project table in the root `README.md`
4. Commit with a meaningful message, e.g. `feat: add LED cube project`

## Commit style

| Prefix | When to use |
|--------|-------------|
| `feat:` | New project or major new feature |
| `fix:` | Bug fix |
| `docs:` | Documentation only |
| `cad:` | New or updated 3D files |
| `fw:` | Firmware change |

## For the kids 👧👦

- You don't need to know everything — ask, experiment, break things (in a safe way!)
- Use comments in code to explain what you were trying to do
- Photos of your builds are always welcome in `docs/`
